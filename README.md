# Dokkaebi

![Dokkaebi](https://github.com/bitleader-dev/DokkaebiTerminal/blob/main/assets/icons/icon.png)

Zed를 기반으로 개발된 AI 코딩 에이전트를 위한 Windows 중심 터미널 작업 공간입니다.

---

### 주요 기능

- **AI 코딩 에이전트 통합** — Claude Code, Gemini CLI, Codex, OpenCode 지원
- **멀티태스킹** — 여러 에이전트 세션을 동시에 운용
- **Zed 기반** — Zed 에디터의 고성능 GPU 렌더링 아키텍처 활용

---
